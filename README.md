# Digital Image Processing (MATLAB)

This repository contains a basic implementation of **Image Processing operations using MATLAB**.  

---

## Objective

To help understand how fundamental image operations work under the hood by building them from scratch.

---

## Features

- Basics of Image Processing
- Histogram Equalization

---

## How to Run

1. Open MATLAB
2. Place the input image file in the same folder as your script  
3. Run the MATLAB code
